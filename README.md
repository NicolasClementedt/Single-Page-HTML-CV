Single-Page HTML CV
A structured, semantic single-page Curriculum Vitae built as part of the roadmap.sh Frontend Developer path.

This project focuses on the core foundations of web development: meaningful structure and metadata, ensuring the document is optimized for both users and search engines before any styling is applied.

🚀 Key Features
Semantic Architecture: Utilizes HTML5 elements for better accessibility and SEO.

SEO & Metadata: Properly configured viewport and description tags for mobile responsiveness and search engine indexing.

Open Graph Protocol: Integrated og:title, og:type, and og:description for professional link previews on social platforms.

Optimized Links: Correct implementation of mailto: and tel: protocols for direct communication.

Clean Organization: Logical separation of skills, education, projects, and professional history.

🛠️ Technologies Used

HTML5

GitHub Pages (Deployment)

📁 Project Sections
Header: Contact details and professional title.

Skills: Core technical stack (TypeScript, NestJS, React, etc.).

Education: Academic background and specialized Bootcamps.

Projects: Featured work, including the NutriLeve full-stack application.

Experience: Current professional roles and volunteer work.

Footer: Direct links to professional profiles (LinkedIn/GitHub).

💡 Learning Objectives

The goal of this specific project was to practice:

Writing clean, self-explanatory HTML without relying on CSS.

Understanding how search engines and crawlers read a webpage.

Setting up a basic deployment pipeline using GitHub.
